About This Project

This project, 'The Moon Awaits', was created as part of my Three.js Bootcamp. During the bootcamp, I learned how to work with 3D models using Blender, export them as .glb files, and use Model Viewer to display them on a website.

The project's objective is to give users a fun and instructive 3D lunar exploration experience. The Moon model used in this project is sourced from NASA's Open Data, allowing users to rotate, zoom, and inspect its surface in detail.

This project demonstrates how to combine Three.js, Model Viewer, and web development to build immersive experiences and reflects my learning experience with 3D web technologies.
